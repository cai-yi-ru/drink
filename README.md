# 今天喝蝦咪？

DEMO：[今天喝蝦咪](https://cai-yi-ru.github.io/drink/)

※備註：因API使用Heroku主機，喚醒需等待時間約30秒

### 專案說明

前端

- 使用 Bootstrap 4 樣式
- 使用axios串接自己設計的 API

後端API

- 使用Express - Node.js Web 應用程式架構
- 使用Express + Firebase 設計 今天喝蝦咪的 API (取得、新增)
- API部署至 Heroku

### ****使用技術****

- vue-axios
- Bootstrap 4
- 自己設計的 API
