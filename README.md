# 今天喝蝦咪？


DEMO：[今天喝蝦咪](https://cai-yi-ru.github.io/drink/)

※備註：因API使用Heroku主機，喚醒需等待時間約30秒

### 專案動機
每次選擇飲料店的時候，時常會有選擇障礙。

### 專案說明

- 使用 Bootstrap 4 樣式
- 使用axios串接自己的 RESTful API

### ****使用技術****

- vue-axios
- Bootstrap 4
- 自己設計的 RESTful API
